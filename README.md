Device configuration for yoshino platform
=========================================

This is the Android 8.0+ device configuration for yoshino platform.

Devices based on yoshino platform:

* Xperia XZ Premium (maple)
* XPeria XZ1 (poplar)
* Xperia XZ1 Compact (lilac)
